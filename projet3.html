Projet front end

Ce projet consiste à la creéation d'un site web spécialisé sur la botanique urbaine. Le site contient une page d’accueil, une page de présentation, des liens vers d’autres sites informatifs et plusieurs catégories accessibles depuis un menu et un footer. Ce projet posséde un seul fichier CSS.

Compétences visées :

  • HTML 5
  • CSS
